type="image" menampilkan gambar icon.png. type="file" memungkinkan user memilih file dari pc sendiri. type="hidden" akan menampilkan nama file yang dipilih pada type="file"

Mengubah nilai min menjadi -50 pada input type="number" akan mengubah batasan nilai input menjadi -50 hingga 10. Sedangkan input type="range" tidak terjadi perubahan apapun.
